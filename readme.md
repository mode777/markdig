# Markdig - Latex Fork
This is a fork of Markdig to render Markdown to Latex

## Usage & Docs
See the [original Latex documentation Repo](https://github.com/lunet-io/markdig).

### Render Latex
 ```csharp
 var latexString = Markdown.ToLatex(markdownString);
 ```
